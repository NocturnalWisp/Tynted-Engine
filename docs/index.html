Welcome to the Tynted-Engine wiki!

There is much for you to learn whether through the API, Docs, or even the examples. I hope you find what you are looking for!

You can find what you need on the side bar. The API is a functional documentation intended as a reference to the programmatic implementation of the explanations demonstrated in the Docs. The docs are useful for knowledge reading and understanding the inner workings of Tynted Engine, and how you can use it to create your game. The Examples can be helpful in understanding practical uses of the engine.

Feel free to look around and find what you need. The docs are intended to go in a natural order to understand the topics at hand, but you of course can use it as a quick reference guide.
